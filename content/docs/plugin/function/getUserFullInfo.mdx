#### 获取用户信息
可以使用 TDLib 的 `getUserFullInfo` 方法来获取用户的详细信息。以下是一个示例代码，展示如何在插件中实现这一功能：

```ts title="UserInfo.ts"
import { getUserFullInfo } from "@TDLib/function/user.ts";

const user = await getUserFullInfo(client, user_id);
```
